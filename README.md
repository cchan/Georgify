Georgify - SE212
===============================

Sublime Text plugin to test proof correctness for the George language used in SE 212.

## Usage
`super+shift+g`to test your proof (Generic)
`cmd+shift+g` to test your proof (Mac)
`windows+shift+g` to test your proof (Mac)
